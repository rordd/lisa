# Τεκμηρίωση στιγμιότυπου έργου και διαλογής (Project Snapshot & Triage)

Χρονικά περιορισμένα στιγμιότυπα κατάστασης έργου για τεκμηρίωση σχεδιασμού και επιχειρησιακές εργασίες.

## Τρέχον στιγμιότυπο

- [../project-triage-snapshot-2026-02-18.md](../project-triage-snapshot-2026-02-18.md)

## Πεδίο εφαρμογής (Scope)

Τα στιγμιότυπα έργου είναι χρονικά περιορισμένες αξιολογήσεις των ανοιχτών PR (Pull Requests), των ζητημάτων (issues) και της κατάστασης της τεκμηρίωσης. Χρησιμοποιήστε τα για να:

- Προσδιορίσετε κενά στην τεκμηρίωση που προκύπτουν από την ανάπτυξη νέων δυνατοτήτων (features).
- Θέσετε προτεραιότητες στη συντήρηση των εγγράφων παράλληλα με τις αλλαγές στον κώδικα.
- Παρακολουθείτε την εξελισσόμενη πίεση από PR/issues με την πάροδο του χρόνου.

Για σταθερή ταξινόμηση τεκμηρίωσης (χωρίς χρονικό περιορισμό), χρησιμοποιήστε το [docs-inventory.md](../docs-inventory.md).
