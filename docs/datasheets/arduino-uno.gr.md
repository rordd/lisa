# Arduino Uno

## Ψευδώνυμα ακροδεκτών (Pin Aliases)

| ψευδώνυμο   | ακροδέκτης |
| :---        | :---       |
| red_led     | 13         |
| builtin_led | 13         |
| user_led    | 13         |

## Επισκόπηση

Το Arduino Uno είναι μια πλακέτα μικροελεγκτή που βασίζεται στον ATmega328P. Διαθέτει 14 ψηφιακούς ακροδέκτες εισόδου/εξόδου (I/O) (0–13) και 6 αναλογικές εισόδους (A0–A5).

## Ψηφιακοί ακροδέκτες

- **Ακροδέκτες 0–13:** Ψηφιακή είσοδος/έξοδος (I/O). Μπορούν να οριστούν ως INPUT (Είσοδος) ή OUTPUT (Έξοδος).
- **Ακροδέκτης 13:** Ενσωματωμένη λυχνία LED (onboard). Συνδέστε τη λυχνία LED στο GND ή χρησιμοποιήστε την ως έξοδο.
- **Ακροδέκτες 0–1:** Χρησιμοποιούνται επίσης για σειριακή επικοινωνία (RX/TX). Αποφύγετε τη χρήση τους εάν χρησιμοποιείτε τη σειριακή θύρα (Serial).

## GPIO

- `digitalWrite(pin, HIGH)` ή `digitalWrite(pin, LOW)` για έξοδο.
- `digitalRead(pin)` για είσοδο (επιστρέφει 0 ή 1).
- Αριθμοί ακροδεκτών στο πρωτόκολλο ZeroClaw: 0–13.

## Σειριακή επικοινωνία (Serial)

- UART στους ακροδέκτες 0 (RX) και 1 (TX).
- Σύνδεση USB μέσω ATmega16U2 ή CH340 (κλώνοι).
- Ρυθμός baud (Baud rate): 115200 για το υλικολογισμικό (firmware) του ZeroClaw.

## Εργαλεία ZeroClaw

- `gpio_read`: Ανάγνωση τιμής ακροδέκτη (0 ή 1).
- `gpio_write`: Ορισμός ακροδέκτη σε υψηλή (1) ή χαμηλή (0) κατάσταση.
- `arduino_upload`: Ο πράκτορας (agent) δημιουργεί πλήρη κώδικα sketch για Arduino. Το ZeroClaw μεταγλωττίζει και μεταφορτώνει τον κώδικα μέσω του arduino-cli. Χρησιμοποιείται για λειτουργίες όπως "δημιουργία μοτίβου καρδιάς" ή προσαρμοσμένα μοτίβα — ο πράκτορας γράφει τον κώδικα, δεν απαιτείται χειροκίνητη επεξεργασία. Ακροδέκτης 13 = ενσωματωμένο LED.
